# Part 1: Neural Network Analysis

## Overview
This directory contains the neural network analysis project for the assignment.

## Project Structure
```
part-1-neural-network-analysis/
├── README.md
├── notebook.ipynb
├── requirements.txt
└── results/
    ├── model_comparison_table.png or .csv
    └── evaluation_outputs.png
```

## Setup Instructions

1. Install required dependencies:
```bash
pip install -r requirements.txt
```

2. Open and run the notebook:
```bash
jupyter notebook notebook.ipynb
```

## Contents

- **notebook.ipynb**: Main Jupyter notebook containing the neural network analysis code
- **requirements.txt**: Python package dependencies
- **results/**: Output directory for model comparisons and evaluation results

## Results
Results and visualizations will be saved to the `results/` directory.
